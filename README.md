# くらしDXサポート

高齢者やITが苦手な方の「デジタル家計の損」を見つけ、相談につなげる公開版アプリです。

## 主な機能

- スマホ料金・サブスクの3ステップ診断
- 年間／月間の節約見込み計算
- デジタル家計簿と見直し履歴
- AI・スマホ・PCなどの相談窓口
- AI導入、スマホ、動画配信、格安SIMを3項目で説明する選び方ガイド
- 電話／訪問／来店相談と予約導線
- PDF保存用の印刷レイアウト
- PC・タブレット・スマートフォン対応
- デジタル庁デザインシステムの公式アイコン・イラスト
- 利用者画面から分離した、運営者向けのセキュリティ方針

## 起動方法

外部ライブラリのインストールは不要です。

```sh
python3 server.py
```

ブラウザで `http://localhost:4173` を開いてください。

## ファイル構成

- `index.html` — 画面とコンテンツ
- `styles.css` — デザインとレスポンシブ対応
- `app.js` — 画面遷移、診断計算、予約操作
- `favicon.svg` — アプリアイコン
- `assets/` — デジタル庁のアイコン・イラスト
- `server.py` — ローカル確認用のセキュリティヘッダー付きサーバー
- `SECURITY.md` — 本番移行を含むセキュリティ・プライバシー設計
- `ASSET-NOTICE.md` — デザインアセットの出典と注意事項

## データの取り扱い

この配布版は外部APIへ接続せず、診断内容を送信・保存しません。運営時の管理要件は [`SECURITY.md`](./SECURITY.md) に分離しています。
